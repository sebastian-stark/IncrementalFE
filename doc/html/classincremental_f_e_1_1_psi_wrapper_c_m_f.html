Detailed Description

template<unsigned int dim, unsigned int spacedim>
class incrementalFE::PsiWrapperCMF< dim, spacedim >

Interface Free Energy wrapping a function defined with the CMF library.

It is assumed that the first parameters of the CMF function are as follows:

parameters[0] ... parameters[spacedim-1] - position vector x
parameters[spacedim] ... parameters[2*spacedim-1] - normal vector n

Further parameters may follow.

Warning
This class is untested so far!
