Numerical parameter between 0 and 1 (see LinearMaterial00<spacedim, spacedim>::method).

Warning
In the very first time step alpha=1.0 is used irrespective of the value provided for alpha! This helps to avoid oscillations of the solution if the initial solution fields are not equilibrated.
