Detailed Description

template<unsigned int dim, unsigned int spacedim>
class incrementalFE::OmegaWrapperCMF< dim, spacedim >

Interface Dissipation potential wrapping a function defined with the CMF library.

It is assumed that the first parameters of the CMF function are as follows:

parameters[0] - time
parameters[1] ... parameters[spacedim] - position vector x
parameters[spacedim+1] ... parameters[2*spacedim] - normal vector n

Further parameters may follow.

Warning
This class is untested so far!
