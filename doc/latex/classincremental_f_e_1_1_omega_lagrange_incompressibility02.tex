\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Lagrange\+Incompressibility02$<$ spacedim $>$}

Class defining an interface related scalar functional with the integrand

$ \omega^\Sigma = J \boldsymbol{f} \cdot \boldsymbol{F}^{-\top} \cdot \boldsymbol{N} \dot{\xi} $,

where $\boldsymbol{F}$ is the deformation gradient, $J$ its determinant, $boldsymbol{f}$ a Lagrangian multiplier vector, $\dot{\xi}$ the rate of a scalar field, and $\boldsymbol{N}$ the unit normal vector.

This is the interfacial counterpart to \mbox{\hyperlink{classincremental_f_e_1_1_omega_lagrange_incompressibility00}{Omega\+Lagrange\+Incompressibility00}}.
