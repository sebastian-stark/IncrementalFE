\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Solution\+Vector\+Type$>$\newline
class incremental\+F\+E\+::\+Manufactured\+Solution\+Spline$<$ Solution\+Vector\+Type $>$}

This class constructs a manufactured solution based on spline interpolation of a discrete dof solution at certain time instants.

This functionality is based on the spline library of T. Kluge (\href{https://kluge.in-chemnitz.de/opensource/spline/}{\texttt{ https\+://kluge.\+in-\/chemnitz.\+de/opensource/spline/}}). If you want to use it, you must pass the path to the header file to cmake by a flag -\/D\+S\+P\+L\+I\+N\+E\+\_\+\+D\+IR=/path/to/spline/header.
