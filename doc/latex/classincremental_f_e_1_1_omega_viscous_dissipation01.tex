\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Viscous\+Dissipation01$<$ spacedim $>$}

Class defining a domain related scalar functional with the integrand

$ \omega^\Omega = \dfrac{A}{2n} (\boldsymbol{d}:\boldsymbol{d})^n $,

where $\boldsymbol{d} = \dfrac{1}{2}\left( \boldsymbol{F}^{-1} \cdot \dot{\boldsymbol{F}} + \dot{\boldsymbol{F}} \cdot \boldsymbol{F}^{-1} \right)$ is the stretching associated with the deformation gradient $\boldsymbol{F}$, and $A$ and $n$ are material parameters.

In order to avoid singular second derivatives,

$h^\Omega_\rho = \dfrac{A_0}{2} \boldsymbol{d}:\boldsymbol{d}$

is used if $\sqrt{\boldsymbol{d}:\boldsymbol{d}} < d_\mathrm{th}$, with $d_\mathrm{th}\ll 1$ and $A_0 = A d_\mathrm{th}^{2(n-1)} $. In practice, $d_\mathrm{th}$ should be chosen such that it is slightly smaller than the maximum of $\sqrt{\boldsymbol{d}:\boldsymbol{d}}$ in the problem. If $n > 1$, $d_\mathrm{th}$ should be set to zero.

This describes an isotropic Norton type creep relation at large strains.
