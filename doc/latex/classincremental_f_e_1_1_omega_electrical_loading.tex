\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Electrical\+Loading$<$ spacedim $>$}

Class defining the following functions

$ \omega^\mathrm{C} = -\dot{\bar{J}}\varphi$ for prescribed current loading (loading type J), $ \omega^\mathrm{C} = -\dot{J}(\varphi - \bar{\varphi})$ for prescribed voltage loading (loading type V), $ \omega^\mathrm{C} = - \dfrac{1}{2 R^\mathrm{el}} \varphi^2$ for discharging through electrical resistance $R^\mathrm{el}$ (loading type R),

The loading type can be changed at any time by changing \mbox{\hyperlink{classincremental_f_e_1_1_omega_electrical_loading_ab46e1a903ac7e5d63ffaf9f4b4822c7a}{Omega\+Electrical\+Loading\+::loading\+\_\+type}}.

Similarly, the prescribed current, the prescribed voltage, and the electrical resistance can be changed at any time by assigning the respective variables \mbox{\hyperlink{classincremental_f_e_1_1_omega_electrical_loading_a93aad8c864f394c9bd96175360db9cb6}{Omega\+Electrical\+Loading\+::j\+\_\+bar}}, \mbox{\hyperlink{classincremental_f_e_1_1_omega_electrical_loading_a695e7c0d11f234fd178bad5ff68ba916}{Omega\+Electrical\+Loading\+::phi}}, and \mbox{\hyperlink{classincremental_f_e_1_1_omega_electrical_loading_a5a8015330aeecc9073215e709d3107ec}{Omega\+Electrical\+Loading\+::\+R\+\_\+el}}.

Ordering of quantities in Scalar\+Functional$<$0, spacedim$>$\+::C \+:~\newline
 \mbox{[}0\mbox{]} $J$~\newline
 \mbox{[}1\mbox{]} $\varphi$
