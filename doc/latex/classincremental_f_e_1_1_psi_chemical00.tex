\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Psi\+Chemical00$<$ spacedim $>$}

Class defining a domain related scalar functional with the integrand

$h^\Omega_\rho = RT c_0 h\left( \dfrac{a c + b}{c_0} \right)$,

where

$ h(x) = \begin{cases} x [ \ln(x)-1] \quad&\mathrm{if}\quad x>\epsilon\\ \epsilon \{ \ln(\epsilon) [ \ln(x) - \ln(\epsilon) + 1] - 1\} \quad&\mathrm{else}, \end{cases} $

$R$ is the gas constant, $T$ the temperature, $c_0$ a reference species concentration, $\mu_0$ a corresponding reference value for the potential, $a$ and $b$ are constants, $c$ the species concentration, and $\epsilon \ll 1$ a regularization parameter to avoid ill-\/conditioning if $c$ is too close to zero.
