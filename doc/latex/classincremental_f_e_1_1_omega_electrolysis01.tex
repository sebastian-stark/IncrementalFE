\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Electrolysis01$<$ spacedim $>$}

Class defining an interface related scalar functional for an idealized description of electrolysis reactions with the integrand

$ \omega^\Sigma = \inf\limits_{\dot{I}}[( \eta^\mathrm{int} - A^{\mathrm{e^-}} F \bar{\varphi} ) \dot{I} + \mathring{\delta}(\dot{I})] $,

where $ \mathring{\delta} = -\dfrac{i_0 R T}{F} \left[ \sqrt{1+\left( \dfrac{F\dot{I}}{i_0} \right)^2} - 1 \right] + RT\dot{I}\mathrm{sinh}^{-1}\left(\dfrac{F\dot{I}}{i_0}\right) + \dfrac{R^\mathrm{el}}{2} \left( F \dot{I} \right)^2 $

where $\eta^\mathrm{int}$ is the driving force for the reaction in the absence of an applied external potential,~\newline
 $F$ is Faraday\textquotesingle{}s constant,~\newline
 $R$ is the gas constant,~\newline
 $T$ is the absolute temperature~\newline
 $A^{\mathrm{e^-}}$ is the number of electrons added to the solution during the reaction,~\newline
 $\bar{\varphi}$ the prescribed external electrical potential,~\newline
 $i_0$ the exchange current density,~\newline
 and $R^\mathrm{el}$ an electrical \char`\"{}interface resistance\char`\"{}.

Ordering of quantities in \textbf{ Scalar\+Functional\+::e\+\_\+sigma} \+:~\newline
\mbox{[}0\mbox{]} $\eta^\mathrm{int}$
