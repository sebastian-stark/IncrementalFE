\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Fluid\+Normal\+Condition01$<$ spacedim $>$}

Class defining a normal boundary condition of the form.

$ \omega^\Sigma = - \boldsymbol{n}\cdot\boldsymbol{d}\cdot\boldsymbol{n} \, J \boldsymbol{f} \cdot \boldsymbol{F}^{-\top} \cdot \boldsymbol{N} $,

where $ \boldsymbol{d} = \mathrm{sym}\left( \nabla \dot{\boldsymbol{v}} \cdot \boldsymbol{F}^{-1} \right) $, $ \boldsymbol{n} = J\,\boldsymbol{F}^{-\top} \cdot \boldsymbol{N} / \sqrt{ \boldsymbol{N} \cdot \boldsymbol{F}^{-\top} \cdot \boldsymbol{F} \cdot \boldsymbol{N} } $ $\boldsymbol{F}$ the deformation gradient,~\newline
 $J$ its determinant, $\dot{\boldsymbol{v}}$ a process variable, $\boldsymbol{f}$ a Lagrangian multiplier.

This enforces incompressibility as an interface condition
