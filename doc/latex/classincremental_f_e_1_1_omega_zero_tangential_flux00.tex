\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Zero\+Tangential\+Flux00$<$ spacedim $>$}

Class defining an interface related scalar functional with the integrand

$ \omega^\Sigma = \boldsymbol{t} \cdot \left[ \left(\dot{\boldsymbol{u}} - \dot{\boldsymbol{v}}\right) \times \boldsymbol{n} \right] + \dfrac{1}{2} \left(\boldsymbol{t} \cdot \boldsymbol{n} \right)^2 $,

where $\dot{\boldsymbol{u}}$ is the velocity of a solid at an interface to a fluid, $\dot{\boldsymbol{v}}$ is the velocity of the fluid at the interface, $\boldsymbol{t}$ is a Lagrangian multiplier, $\boldsymbol{n} = J \boldsymbol{F}^{-\top} \cdot \boldsymbol{N}$ is the normal vector on the interface in the deformed configuration, with $\boldsymbol{F}$ being the deformation gradient, $J$ its determinant, and $\boldsymbol{N}$ the normal vector on the interface in the reference configuration.

This constrains the tangential relative velocity between a fluid and a solid surface to zero. In order to regularize the problem, a positive definite term is added for the normal component of $\boldsymbol{t}$.
