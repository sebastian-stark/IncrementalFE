\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Viscous\+Dissipation02$<$ spacedim $>$}

Class defining a domain related scalar functional with the integrand

$ \omega^\Omega = \dfrac{J \eta}{2} (\boldsymbol{d}:\boldsymbol{d}) $,

where $\boldsymbol{d} = \dfrac{1}{2}\left( \boldsymbol{F}^{-1} \cdot \nabla \boldsymbol{v} + \nabla \boldsymbol{v} \cdot \boldsymbol{F}^{-1} \right)$ is the stretching, $\boldsymbol{F}$ is the deformation gradient associated with the background solid skeleton, $eta$ is the viscosity, and $J = \det\boldsymbol{F}$.

This describes a viscous fluid.
