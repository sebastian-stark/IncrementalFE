\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim, unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Wrapper\+C\+M\+F$<$ dim, spacedim $>$}

Interface Dissipation potential wrapping a function defined with the C\+MF library.

If \mbox{\hyperlink{classincremental_f_e_1_1_omega_wrapper_c_m_f_a1660368a9fd39f03d33133878bf61544}{Omega\+Wrapper\+C\+M\+F\+::use\+\_\+param}} is true, it is assumed that the first seven parameters of the C\+MF function are as follows\+:~\newline


parameters\mbox{[}0\mbox{]} -\/ time~\newline
 parameters\mbox{[}1\mbox{]} ... parameters\mbox{[}3\mbox{]} -\/ position vector x~\newline
 parameters\mbox{[}4\mbox{]} ... parameters\mbox{[}6\mbox{]} -\/ normal vector n~\newline


In two dimensions, the third components of x and n are filled with zeroes.

Further parameters may follow and are defined through \mbox{\hyperlink{classincremental_f_e_1_1_omega_wrapper_c_m_f_a9756f3b67d9a7ff37cb8c3590bde3ffa}{Omega\+Wrapper\+C\+M\+F\+::param\+\_\+fun}}.

This class is untested so far! 
\end{DoxyWarning}
