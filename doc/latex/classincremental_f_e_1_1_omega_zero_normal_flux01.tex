\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Zero\+Normal\+Flux01$<$ spacedim $>$}

Class defining an interface related scalar functional with the integrand

$ \omega^\Sigma = t \left(\dot{\boldsymbol{u}} - \dot{\boldsymbol{v}}\right) \cdot \boldsymbol{n} $,

where $\dot{\boldsymbol{u}}$ is the velocity of a solid at an interface to a fluid, $\dot{\boldsymbol{v}}$ is the velocity of the fluid at the interface, $t$ is a Lagrangian multiplier, $\boldsymbol{n} = J \boldsymbol{F}^{-\top} \cdot \boldsymbol{N}$ is the normal vector on the interface in the deformed configuration, with $\boldsymbol{F}$ being the deformation gradient, $J$ its determinant, and $\boldsymbol{N}$ the normal vector on the interface in the reference configuration.

This constrains the normal relative velocity between a fluid and a solid surface to zero.
