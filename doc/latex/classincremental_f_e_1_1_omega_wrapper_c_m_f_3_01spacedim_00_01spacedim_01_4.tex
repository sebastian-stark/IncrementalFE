\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class incremental\+F\+E\+::\+Omega\+Wrapper\+C\+M\+F$<$ spacedim, spacedim $>$}

Volume Dissipation potential wrapping a function defined with the C\+MF library.

If \mbox{\hyperlink{classincremental_f_e_1_1_omega_wrapper_c_m_f_3_01spacedim_00_01spacedim_01_4_a823cb1a77890e6f180a6a688da89a4bd}{Omega\+Wrapper\+C\+M\+F$<$spacedim,spacedim$>$\+::use\+\_\+param}} is true, it is assumed that the first four parameters of the C\+MF function are as follows\+:~\newline


parameters\mbox{[}0\mbox{]} -\/ time~\newline
 parameters\mbox{[}1\mbox{]} ... parameters\mbox{[}3\mbox{]} -\/ position vector x~\newline


In two dimensions, the third component of x is filled with zeroes.

Further parameters may follow and are defined through \mbox{\hyperlink{classincremental_f_e_1_1_omega_wrapper_c_m_f_3_01spacedim_00_01spacedim_01_4_aef0e77db45a89ad373648ff9fb3cc377}{Omega\+Wrapper\+C\+M\+F$<$spacedim,spacedim$>$\+::param\+\_\+fun}}.

This class is untested so far! 
\end{DoxyWarning}
